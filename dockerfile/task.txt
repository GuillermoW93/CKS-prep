Create this insecure Dockerfile:

FROM ubuntu:latest
USER root
RUN apt-get update && apt-get install -y curl
ADD app.tar.gz /opt/app/
CMD ["/opt/app/run.sh"]

Create this insecure deployment.yaml:

apiVersion: apps/v1
kind: Deployment
metadata:
  name: vulnerable-app
spec:
  replicas: 1
  selector:
    matchLabels:
      app: vuln
  template:
    metadata:
      labels:
        app: vuln
    spec:
      containers:
      - name: app
        image: myregistry/vulnerable-app:latest
        securityContext:
          privileged: true

----

The Task:

Rewrite the Dockerfile to use a specific, pinned base image tag (not latest), run as a non-root user, and use COPY instead of ADD.

Rewrite the deployment.yaml to remove privileged access and drop all Linux capabilities (ALL).
